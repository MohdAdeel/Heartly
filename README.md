# ❤️ Heartly - React Native - Match Making App

**Everyone Deserves a Special Someone.**

Heartly is a modern matchmaking app designed for people who are serious about relationships. It goes beyond swipes—focusing on compatibility, authenticity, and genuine connections.

---

## ✨ About Heartly

Heartly is built with one goal in mind:
**to help people find real, lasting relationships in a simple and elegant way.**

Whether you're looking for a life partner or a meaningful connection, Heartly provides a clean, intuitive experience that prioritizes quality over quantity.

---

## 🚀 Features

* 💖 Smart matchmaking based on user preferences  
* 🔒 Secure authentication and user data handling  
* 📸 Profile creation with high-quality image uploads  
* 🧑‍💼 Face verification for authentic profiles  
* 🛡️ Anti-fake & duplicate account detection  
* 🔔 Real-time push notifications (FCM)  
* 💬 Smooth and intuitive user interaction experience  
* ❤️ Like, match, and connect instantly  
* 🔍 Advanced search & filtering options  
* 📍 Location-based matching  
* ⚡ Fast and scalable Node.js backend  
* ☁️ Supabase-powered database & authentication  
* 🕒 Activity tracking (recently active users)  
* 🚫 Block & report system for safety  
* 🎯 Personalized recommendations  

---

## 🛠️ Tech Stack

**Frontend**

* TypeScript
* (React Native)

**Backend**

* Node.js
* REST APIs

**Database & Auth**

* Supabase (Authentication + PostgreSQL Database)

**Notifications**

* Firebase Cloud Messaging (FCM)

**Face Verification**

* Didit
